# HypeIT
Markedsføring og Salg samarbeidsprosjekt.


Et samarbeidsprosjekt med hele klassa, vi har 4 grupper:
 - Front-end
 - Back-end
 - Design
 - Markedsføring

Vi skal lage en fullstendig online bedrift bortett bra at vi ikke egentlig skal selge noe.
